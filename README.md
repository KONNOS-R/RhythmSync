RhythmSync
CLI audio player that plays music while displaying lyrics (LRC) in sync.

For installation run:
pip install --index-url https://test.pypi.org/simple/ \
--extra-index-url https://pypi.org/simple \
--upgrade RhythmSync

After installation launch the app named 'rhythmsync' using python and simply enter the path where your music file is located.

(In order for the lyrics feature to work your files need to have embedded LRC data)
