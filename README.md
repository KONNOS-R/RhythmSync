-----------------------------RhythmSync-----------------------------
CLI audio player that plays music while displaying lyrics (LRC) in sync.

For installation run:
pip install --index-url https://test.pypi.org/simple/ \
--extra-index-url https://pypi.org/simple \
--upgrade RhythmSync

After installation simply create the 'AddMusicHere' directory at the same location as 'rhythmsync' and add your music there. After that call the name of your music file in the cli application named 'rhythmsync' which can be executed with python!

(In order for the lyrics feature to work your files need to have embedded LRC data)
